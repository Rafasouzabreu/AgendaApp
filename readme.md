# AgendaApp

O AgendaApp é uma solução de software desenvolvida com o objetivo de oferecer uma ferramenta de gerenciamento de usuários e tarefas eficiente e intuitiva. Utilizando uma combinação de tecnologias modernas e padrões de design, este projeto visa proporcionar uma experiência de usuário excepcional, ao mesmo tempo que mantém a qualidade do código e facilita a manutenção.

## Tecnologias Utilizadas

- **ASP.NET API:** Utilizado para construir os serviços RESTful que servem os dados para os clientes.
- **ASP.NET MVC:** Framework para construção das páginas web, seguindo o padrão Model-View-Controller.
- **Entity Framework:** ORM utilizado para abstrair o acesso ao banco de dados.
- **Dapper:** Biblioteca utilizada para operações de banco de dados que exigem alto desempenho.
- **AutoMapper:** Utilizado para mapear entidades de domínio para DTOs de forma eficiente.
- **JWT (JSON Web Token):** Implementa autenticação e autorização baseadas em tokens.
- **DDD (Domain-Driven Design):** Abordagem utilizada para o design da arquitetura do software, focando no domínio do negócio.
- **XUnit:** Framework de testes utilizado para realizar testes unitários e de integração.
- **Bootstrap:** Framework de front-end para desenvolvimento de interfaces web responsivas e mobile-first.

## Pré-Requisitos

Para executar o AgendaApp em seu ambiente de desenvolvimento, você precisará ter instalado:

- .NET 8.0 SDK ou superior
- SQL Server
- Visual Studio 2022
